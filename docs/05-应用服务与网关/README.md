# 应用服务与网关

## 模块概述

本模块负责实现基于Kong/Istio的多租户API网关和服务网格方案。

## 目录结构

```
05-应用服务与网关/
├── README.md                    # 本文档
├── 5.1-多租户网关与服务.md      # Kong网关配置
└── 部署配置/
    ├── kong-gateway.yaml
    └── tenant-routing.yaml
```

## 核心功能

### 5.1 多租户网关与服务
- Kong API网关部署
- 租户路由规则配置
- 租户级限流策略
- 多实例负载均衡
- 微服务间mTLS通信

## 技术栈

- **API网关**: Kong 3.x
- **服务网格**: Istio 1.18.x
- **负载均衡**: Kong + kube-proxy
